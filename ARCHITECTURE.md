# Aura X Architecture

## Overview
Aura X is a Web3 Financial Operating System built on React, Vite, and Tailwind CSS.
It supports a multi-chain architecture for institutional and public scale.

## Core Layers
- **Frontend**: React 18 (SPA)
- **Styling**: Tailwind CSS, Headless UI components
- **State Management**: React Hooks & Context
- **Web3 Layer**: Multi-chain support (Ethereum, Polygon, Base, Arbitrum, BNB Chain)
- **Database/Auth**: Firebase (Firestore, Authentication)

## Observability & Security
- Integrated error monitoring and performance tracking.
- Content Security Policy (CSP) headers via Nginx.
